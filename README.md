Browser-based desktop environment inspired by webOS, i3wm, and Arch Linux. Delivers an interactive, desktop-like experience in the browser with a focus on window management, keyboard-driven navigation, and a polished UI.

Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- React 19
- React Icons
- Vercel Analytics

Features
- Interactive desktop-style interface
- Tiling and floating window management
- Draggable windows
- Workspace switching
- Keyboard shortcuts for fast navigation
- Application launcher and power search
- Customizable background themes
- Built-in apps: terminal, browser, YouTube, paint, text editor, settings

Setup
- Install dependencies:
npm install
- Start the development server:
npm run dev
